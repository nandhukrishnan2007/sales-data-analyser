 Sales Data Analyzer
This project analyzes sales and profit data using Python and Pandas.

 Features
- Data cleaning (handling missing values)
- Data analysis (average, total, max)
- Data visualization (bar charts)
- Insights (top & lowest performer)
- Export cleaned dataset

  Tools Used
- Python
- Pandas
- Matplotlib

 Output
- Sales & profit charts
- Summary report in terminal
- Cleaned CSV file

 Files
- data.csv → raw data  
- cleaned_data.csv → processed data  
- analysis.py → main code  

 This is my first data analytics project.
